GetDialogUnreadMarksRequest

Only users can use this request. See code examples.

---functions---
messages.getDialogUnreadMarks#21202222 flags:# parent_peer:flags.0?InputPeer = Vector<DialogPeer>

Returns

A list of the following type is returned.

DialogPeer

This type can be an instance of either:

DialogPeerDialogPeerFolder

Parameters

parent_peerInputPeerThis argument defaults to None and can be omitted. Anything entity-like will work if the library can find its Input version (e.g., usernames, Peer, User or Channel objects, etc.).

Known RPC errors

This request can't cause any RPC error as far as we know.

Example

from telethon.sync import TelegramClient
from telethon import functions, types

with TelegramClient(name, api_id, api_hash) as client:
    result = client(functions.messages.GetDialogUnreadMarksRequest(
        parent_peer='username'
    ))
    for x in result:
        print(x)
