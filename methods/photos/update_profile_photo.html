UpdateProfilePhoto

Both users and bots can use this request.

---functions---
photos.updateProfilePhoto#09e82039 fallback:flags.0?true bot:flags.1?InputUser id:InputPhoto = photos.Photo

Returns

photos.Photo

Parameters

fallbacktrueoptional
botInputUseroptional
idInputPhotorequired

Example

▶ Minimal (required fields only) 
from ferogram import raw, types

result = await client(raw.functions.photos.UpdateProfilePhoto(
    id=types.InputPhoto()
))
print(result)
▶ Full (all fields) 
from ferogram import raw, types

result = await client(raw.functions.photos.UpdateProfilePhoto(
    fallback=True,
    bot='username',
    id=types.InputPhoto()
))
print(result)
