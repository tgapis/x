SaveAutoSaveSettings

Both users and bots can use this request.

---functions---
account.saveAutoSaveSettings#d69b8361 users:flags.0?true chats:flags.1?true broadcasts:flags.2?true peer:flags.3?InputPeer settings:AutoSaveSettings = Bool

Returns

Example

▶ Minimal (required fields only) 
from ferogram import raw, types

result = await client(raw.functions.account.SaveAutoSaveSettings(
    settings=types.AutoSaveSettings()
))
print(result)
▶ Full (all fields) 
from ferogram import raw, types

result = await client(raw.functions.account.SaveAutoSaveSettings(
    users=True,
    chats=True,
    broadcasts=True,
    peer='username',
    settings=types.AutoSaveSettings()
))
print(result)
