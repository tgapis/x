UpdateConnectedBot

Both users and bots can use this request.

---functions---
account.updateConnectedBot#66a08c7e deleted:flags.1?true rights:flags.0?BusinessBotRights bot:InputUser recipients:InputBusinessBotRecipients = Updates

Returns

Updates

Parameters

deletedtrueoptional
rightsBusinessBotRightsoptional
botInputUserrequired
recipientsInputBusinessBotRecipientsrequired

Example

▶ Minimal (required fields only) 
from ferogram import raw, types

result = await client(raw.functions.account.UpdateConnectedBot(
    bot='username',
    recipients=types.InputBusinessBotRecipients()
))
print(result)
▶ Full (all fields) 
from ferogram import raw, types

result = await client(raw.functions.account.UpdateConnectedBot(
    deleted=True,
    rights=types.BusinessBotRights(),
    bot='username',
    recipients=types.InputBusinessBotRecipients()
))
print(result)
