AcceptAuthorization

Both users and bots can use this request.

---functions---
account.acceptAuthorization#f3ed4c73 bot_id:long scope:string public_key:string value_hashes:Vector<SecureValueHash> credentials:SecureCredentialsEncrypted = Bool

Returns

Bool

Parameters

bot_idlongrequired
scopestringrequired

value_hashesVector<SecureValueHash>required
credentialsSecureCredentialsEncryptedrequired

Example

from ferogram import raw, types

result = await client(raw.functions.account.AcceptAuthorization(
    bot_id=-12398745604826,
    scope='some string here',
    public_key='some string here',
    value_hashes=types.Vector<SecureValueHash>(),
    credentials=types.SecureCredentialsEncrypted()
))
print(result)
