GetPaymentForm

Both users and bots can use this request.

---functions---
payments.getPaymentForm#37148dbb invoice:InputInvoice theme_params:flags.0?DataJSON = payments.PaymentForm

Example

▶ Minimal (required fields only) 
from ferogram import raw, types

result = await client(raw.functions.payments.GetPaymentForm(
    invoice=types.InputInvoice()
))
print(result)
▶ Full (all fields) 
from ferogram import raw, types

result = await client(raw.functions.payments.GetPaymentForm(
    invoice=types.InputInvoice(),
    theme_params=types.DataJSON()
))
print(result)
