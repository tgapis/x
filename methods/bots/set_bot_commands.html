SetBotCommands

Both users and bots can use this request.

---functions---
bots.setBotCommands#0517165a scope:BotCommandScope lang_code:string commands:Vector<BotCommand> = Bool

Example

from ferogram import raw, types

result = await client(raw.functions.bots.SetBotCommands(
    scope=types.BotCommandScope(),
    lang_code='some string here',
    commands=types.Vector<BotCommand>()
))
print(result)
