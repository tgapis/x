EditAccessSettings

Both users and bots can use this request.

---functions---
bots.editAccessSettings#31813cd8 restricted:flags.0?true bot:InputUser add_users:flags.1?Vector<InputUser> = Bool

Returns

Bool

Parameters

restrictedtrueoptional
botInputUserrequired
add_usersVector<InputUser>optional

Example

▶ Minimal (required fields only) 
from ferogram import raw, types

result = await client(raw.functions.bots.EditAccessSettings(
    bot='username'
))
print(result)
▶ Full (all fields) 
from ferogram import raw, types

result = await client(raw.functions.bots.EditAccessSettings(
    restricted=True,
    bot='username',
    add_users=types.Vector<InputUser>()
))
print(result)
