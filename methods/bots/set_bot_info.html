SetBotInfo

Both users and bots can use this request.

---functions---
bots.setBotInfo#10cf3123 bot:flags.2?InputUser lang_code:string name:flags.3?string about:flags.0?string description:flags.1?string = Bool

Returns

Bool

Parameters

botInputUseroptional
lang_codestringrequired
namestringoptional
aboutstringoptional
descriptionstringoptional

Example

▶ Minimal (required fields only) 
from ferogram import raw, types

result = await client(raw.functions.bots.SetBotInfo(
    lang_code='some string here'
))
print(result)
▶ Full (all fields) 
from ferogram import raw, types

result = await client(raw.functions.bots.SetBotInfo(
    bot='username',
    lang_code='some string here',
    name='some string here',
    about='some string here',
    description='some string here'
))
print(result)
