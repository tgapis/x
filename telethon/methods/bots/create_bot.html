CreateBotRequest

Both users and bots may be able to use this request. See code examples.

---functions---
bots.createBot#e5b17f2b flags:# via_deeplink:flags.0?true name:string username:string manager_id:InputUser = User

Returns

User

This type can be an instance of either:

UserUserEmpty

Parameters

namestring
usernamestring
manager_idInputUserAnything entity-like will work if the library can find its Input version (e.g., usernames, Peer, User or Channel objects, etc.).
via_deeplinkflagThis argument defaults to None and can be omitted.

Known RPC errors

This request can't cause any RPC error as far as we know.

Example

from telethon.sync import TelegramClient
from telethon import functions, types

with TelegramClient(name, api_id, api_hash) as client:
    result = client(functions.bots.CreateBotRequest(
        name='some string here',
        username='some string here',
        manager_id='username',
        via_deeplink=True
    ))
    print(result.stringify())
